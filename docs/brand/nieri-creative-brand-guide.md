Source board: `docs/brand/Branding Board_NC.png`

## Logo

- Primary logo: black Nieri Creative mark on white.
- App-ready crop: `docs/brand/nieri-logo-black.png`

## Fonts

- Display / heading: League Spartan
- Body / secondary: Garet

Until exact licensed font files are added to the Expo project, the app uses native system fallbacks that approximate the weight and clarity of the board. Add font files under `mobile/assets/fonts/` and wire them through `expo-font` when the licensed Garet files are available.

## Color Palette

| Token | Hex | Use |
| --- | --- | --- |
| Charcoal | `#2c2c2c` | Primary text, dark surfaces, primary buttons |
| White | `#FFFFFF` | Primary background and card surfaces |
| Nieri Yellow | `#FFCF00` | Brand accent, selected states, hero highlight |
| Warm Gray | `#a09b88` | Muted brand accent, borders, supporting UI |

## App Usage Notes

- Use yellow as a confident accent rather than a full-screen default.
- Keep UI clean, high-contrast, and professional for real estate agents.
- Prefer black/charcoal typography with yellow highlights and restrained warm-gray dividers.
- Avoid off-brand blue, green, or gold accent palettes unless communicating status.
